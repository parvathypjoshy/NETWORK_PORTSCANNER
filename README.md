# NETWORK_PORTSCANNER
TCP port scanner written in Python

This project is a basic TCP port scanner written in Python.
It checks whether specific ports on a target host are open or closed.

## Purpose
This project was created for learning networking and cybersecurity basics,
specifically how port scanning works.

## Features
- Scan specific TCP ports
- Works with domain names and IP addresses
- Simple and beginner-friendly

## Technologies Used
- Python 3
- Socket library
